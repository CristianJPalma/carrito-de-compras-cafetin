# Alcance del Proyecto — Sistema de Gestión de Pedidos (Cafetín SENA Industrial)

## 1. Objetivos
Optimizar el proceso de atención del Cafetín del SENA Industrial mediante un sistema básico de gestión de pedidos que reduzca los tiempos de espera y minimice errores en la toma de pedidos.

---

## 2. Entregables
- Sistema de gestión de pedidos con **catálogo de productos**.
- Funcionalidad de **carrito de compras** y **confirmación de pedidos**.
- Módulo básico para la **visualización** y el **cambio de estado** de pedidos por parte del Cafetín.

---

## 3. Exclusiones
- Procesamiento de **pagos en línea**.
- Gestión **avanzada de inventarios**.
- Integración con **sistemas externos o contables**.
- Aplicaciones móviles **nativas**.

---

## 4. Limitaciones
- Proyecto desarrollado con **fines académicos**.
- Tiempo y recursos **limitados** según el cronograma del SENA.
